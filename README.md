##  自訂主題 Theming  ##

我採用之前使用的顏色來做這個程式的主題。
- 主色——黃色：`#F2B705`
- 次色——紫色：`#42378C`
- 第三色——紅色：`#F20505`
- 背景色——暖白色： `#959088`

並且透過Google提供的Material Design Palette Generator產生主題的顏色，也方便我們在撰寫Flutter程式時可以符合Material Design的規範。
詳情我放在[Theming的詳細介紹](docs/000-theming.md)。
